Installation

(For UWP release only)

  1. Download latest UWP release.
  2. Open .zip.
  3. Extract .zip.
  4. Run Install.ps1 with PowerShell.
  5. If prompted to install certificate, continue. Otherwise, simply respond "Yes" to all prompts.
  6. "Yes" to all prompts.
  7. Finish installation.
